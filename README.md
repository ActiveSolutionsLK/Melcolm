# Melcolm

## Folder Structure

- <ins>**admin**</ins>: Contains files related to the administration or backend of the application, such as admin panel scripts or configuration files.
- <ins>**assets**</ins>: Holds static files like images, stylesheets (CSS), and client-side scripts (JavaScript) used in the project.
- <ins>**config**</ins>: Contains configuration files for the application, such as database configurations, environment settings, or constants.
- <ins>**includes**</ins>: Contains reusable code snippets or files that are included in multiple pages of the application, such as header, footer, or sidebar.
- <ins>**templates**</ins>: Contains template files for the application's views or pages, which are used to generate the final HTML output.
